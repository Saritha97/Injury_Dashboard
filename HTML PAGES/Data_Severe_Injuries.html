Introduction

The file deals with processing the OSHA Injury Database by cleaning and transforming various columns, including EventDate, Primary NAICS, Nature, Part of Body, and Event. For each of these columns, you are creating new categorical variables based on patterns found in the data, such as classifying the NAICS code into industries, the nature of the injury, the body part affected, and the type of event. You are also converting the EventDate into a year column and filtering out rows with missing or invalid NAICS codes.

Step 1: Load required libraries and files

The library ‘tidyverse’ is used in this assignment because it holds all the core packages used for data manipulation, data cleaning and visualization like dplyr, stringr, ggplot2 and many more.

The below piece of code is used for dropping NA and irrelevant values from Primary NAICS column.

injury_data <- injury_data[!is.na(injury_data$`Primary NAICS`) & injury_data$`Primary NAICS` != 999999, ]

injury_data$event_date <- as.Date(injury_data$EventDate, format = "%m/%d/%Y")

The code cleans and selects relevant columns from the injury_data dataset, making it more manageable for analysis. By renaming the columns and filtering out rows with an “Uncategorized” industry, it improves both the clarity and the quality of the data for further processing.

injury_data <- injury_data |>
  select(
    `year`,
    `industry`,
    `Employer`,
    `City`,
    `State`, 
    `Hospitalized`, 
    `Amputation`,
    `nature_category`,
    `event_category`,
    `source_category`,
    `body_part`,
    `Final Narrative`
  )|>
  rename(
    employer = `Employer`,
    city = `City`,
    state = `State`, 
    hospitalization = `Hospitalized`, 
    amputation = `Amputation`,
    description = `Final Narrative`
  )
injury_data <- injury_data |>
  filter(industry != "Uncategorized")
